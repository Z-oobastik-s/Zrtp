# Zrtp - Плагин случайной телепортации для Minecraft

Zrtp (Random Teleport) - это мощный плагин для серверов Minecraft, который позволяет игрокам телепортироваться в случайную безопасную точку мира. 

## Возможности

- **Случайная телепортация** в безопасное место в любом мире
- **Поддержка всех миров** (обычный мир, Нижний мир/Ад, Край)
- **Настраиваемые параметры телепортации** для каждого мира (радиус, задержка, кулдаун)
- **Визуальные и звуковые эффекты** во время телепортации
- **Система наград** после телепортации (настраиваемые предметы и шансы)
- **Интеграция с экономикой** (через Vault) для платной телепортации
- **Гибкие настройки разрешений** (permissions)
- **Поддержка нескольких языков** (русский, английский)
- **Эффекты после телепортации** (молния, голод, урон и другие)

## Команды

- `/rtp` - Телепортация в случайное место
- `/wild` - Альтернативная команда для телепортации
- `/rtp <игрок>` - Телепортация другого игрока
- `/rtp <игрок> <мир>` - Телепортация игрока в конкретный мир
- `/rtp reload` - Перезагрузка конфигурации плагина

## Разрешения (Permissions)

- `rtp.use` - Разрешает использовать команду телепортации (по умолчанию: true)
- `rtp.bypass` - Игнорирует кулдаун телепортации (по умолчанию: op)
- `rtp.bypass.delay` - Игнорирует задержку перед телепортацией (по умолчанию: op)
- `rtp.other` - Разрешает телепортировать других игроков (по умолчанию: op)
- `rtp.reload` - Разрешает перезагружать плагин (по умолчанию: op)
- `rtp.notme` - Запрещает другим телепортировать этого игрока (по умолчанию: false)
- `rtp.nocooldown` - Позволяет игнорировать время перезарядки между телепортациями (по умолчанию: op)
- `rtp.*` - Все разрешения плагина (по умолчанию: op)

## Установка

1. Скачайте плагин из раздела [Releases](https://github.com/Z-oobastik-s/Zrtp/releases)
2. Поместите .jar файл в папку plugins вашего сервера
3. Перезапустите сервер или используйте плагин для загрузки новых плагинов
4. Настройте конфигурацию в файле `plugins/Zrtp/config.yml`

## Конфигурация

Плагин имеет обширные возможности настройки в файле `config.yml`. Вы можете настроить:

- Параметры телепортации для каждого мира (радиус, задержка, кулдаун)
- Стоимость телепортации
- Визуальные и звуковые эффекты
- Эффекты после телепортации
- Система наград
- Много других параметров

## Языки

Плагин поддерживает несколько языков:
- Русский (ru_RU)
- Английский (en_US)

Вы можете выбрать язык в конфигурации, изменив параметр `language` в файле `config.yml`.
